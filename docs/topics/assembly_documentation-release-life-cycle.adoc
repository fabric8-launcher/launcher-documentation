= Documentation release life cycle

The documentation releases occur periodically, and the work is tracked in JIRA.
The person responsible is typically the Documentation Product Manager (DPM), but any member of the team with sufficient privileges can perform the respective tasks.
The prerequisites are included separately with each task.
